1. What is VizPort One?
VizPort One is the first tool of the Visualization Portal for for Biological Data. It is an ongoing data visualization project that obtains protein variation data from the Ensembl API and organizes and visualizes it in a simple, yet very effective and easy to use way.

2. What proteins can be currently found on the platform?
There is a small amount of proteins that can be currently found on the platform. Historically relevant proteins such as ERBB-2 can be visualized, as well as non-cancer related proteins such as IRF8, CCR5 and PLD3. The number of proteins will continue to increase with the development of this tool. If you would like to suggest a protein, please contact us.

3. How to visualize a protein?
To visualize a protein from the home screen, simply click on the name of the protein that you want to visualize. If you are currently visualizing a protein and would like to load a different one, click on the protein name on the top navigation bar.

4. How to navigate within a protein?
Once you have selected a protein, you can see its name and description on the top navigation bar. Below you will find the protein range bar. This bar describes the range of amino acids that are currently displayed. By default, this range displays the complete protein.

To only view a portion of the protein in order to enhance the level of detail, click and drag one of the two circular handles and the the graphics will update automatically.

5. How to interact with the domains graphic?
The domains section contains annotated domains of several sources. These sources can be viewed below the text “Domains” in a horizontal arrangement paired with a unique color.

By hovering over any of the domain sources, the domains graphic will update, and the source of interest will be highlighted, while the rest of the sources will be almost invisible, allowing the user to focus on the required information. By clicking on a domain, a dialogue box will be displayed with further options.

6. How to interact with the exons graphic?
The exons graphic behaves similarly to the domains graphic. Hover with the mouse pointer over one of the colored lines that represent each alternating exon of the protein to view specific information about it. By clicking on the same line, a dialogue box will be displayed with further options.

7. How to interact with the variations graphic?
The variations graphic shows all the annotated variations for a specific protein. Every curved line in the graphic represents one single variation (i.e. an rs or a somatic variation). Similarly, every color in the graphic represents a type of variation (e.g. inframe deletion, missense variant, etc).

Please note that there is a scale on the top of this graphic that serves as a permanent point of reference and indicates the number of the amino acid in the transcription. By hovering over any of the lines with the mouse pointer, detailed information about a specific variant will be displayed in the form of a tooltip.

In the same fashion, hovering with the mouse pointer over one of the colored bars at the bottom of the graphic will highlight that specific type of variations and decrease the opacity of the rest.

8. What data can be viewed about a specific protein?
There’s detailed data that can be obtained from any protein in VizPort.

For the domain features, the user can see its ID (e.g. PF03494), source (e.g. PANTHER), the name of the domain (e.g. Amyloidogenic glycoprotein, amyloid-beta peptide) and its position (e.g. aa 680 - aa 705).

For the exon features, the user can se its ID (e.g. ENSE00003527715), its start and end phases and its position (e.g. aa 221 - aa 289).

For the variation features, if available, the user can see the ID of the variation (e.g. rs201794320), its type (e.g. Synonymous Variant), position (e.g. aa 555), codon (e.g. gaT/gaC), alleles (e.g. A/G) and residues (e.g. D/D).

Also, by clicking on the hyperlink “Additional Information”, the tool with load a new tab on the browser that redirects to the PharmGKB or COSMIC portals for rs and COSMIC IDs respectively. If the ID has been registered in any of these portals, the user will find expanded pharmacogenomic information.

9. How can I save a specific domain, exon or protein variation?
To save a feature, click on it. A dialogue box will appear with a save button on the bottom left corner. Click the save button.

10. How can I review and manage my saved features?
You can find the saved features at the bottom of the page, below the graphics under a dark grey section entitled “Annotation”. There are four tabs named “all”, “domains”, “exons” and “variations”.

You can review the data of the feature by pressing the arrow on the left of the box and the “X” on the right to delete it.

11. How can I export the saved data for further use?
To export the data as a CSV file, press the Download button on the right of the annotations bar. You will be able to download an editable file with all the annotations.

12. How can I contact the development team?
If you wish to suggest a feature, improvement or simply leave a comment, please write to info@vizport.io
